# Lab 21: Cloud Storage Cost Calculator (OCP)

Цей консольний додаток розраховує вартість хмарного сховища, демонструючи принцип **Open/Closed Principle (OCP)**. 

Система дозволяє додавати нові тарифні плани (стратегії) без зміни основного коду сервісу розрахунків (`CloudStorageService`).

## Використані патерни
* **Strategy:** Кожен тарифний план (Personal, Business, Enterprise, Student) — це окремий клас, що реалізує інтерфейс `IPricingStrategy`.
* **Factory Method:** Клас `PricingStrategyFactory` відповідає за створення потрібної стратегії на основі вибору користувача.

## Доступні тарифи
1.  **Personal:** Низька ціна за ГБ, кількість користувачів не впливає.
2.  **Business:** Вища ціна за ГБ + плата за кожного користувача.
3.  **Enterprise:** Оптова ціна за ГБ + висока ціна за підтримку користувачів + податок.
4.  **Student:** Пільговий тариф (приклад розширення функціоналу).
